# defaultdots

Cópia **original / de fábrica** de dotfiles, mantida como referência para
"resetar" para o estado limpo quando necessário.

| Arquivo | O que é |
|---|---|
| `.zshrc` | `~/.zshrc` padrão do oh-my-zsh (tema `robbyrussell`, sem customizações), antes das personalizações que vivem em [`../dotfiles`](../dotfiles). |

Use como base de comparação ou ponto de partida — **não** é o arquivo usado no
dia a dia (esse é o de [`../dotfiles`](../dotfiles)).
